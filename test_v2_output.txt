### V2 vs V1 KEY DIFFERENCES ###
------------------------------------------------------------
1. ID Parameter:
   V1: outbound_id / inbound_id
   V2: pearl_id

2. Pearl Entity:
   V1: Separate Inbound and Outbound classes
   V2: Unified Pearl class

3. Lead Operations:
   V1: pearl.Outbound.add_lead(outbound_id, ...)
   V2: pearl.Outbound.add_lead(pearl_id, ...)

4. Analytics:
   V1: pearl.Outbound.get_analytics(outbound_id, ...)
   V2: pearl.Pearl.get_analytics(pearl_id, ...)

5. Not Available in V2:
   - Inbound.get_all() >> Use Pearl.get_all()
   - Outbound.get_all() >> Use Pearl.get_all()
   - Outbound.make_call() >> Different mechanism
   - Outbound.get_call_requests() >> Not in V2

6. New in V2:
   - Pearl.get_all() - Get all pearls (inbound + outbound)
   - Pearl.get_analytics() - Unified analytics
   - Enhanced sentiment analysis
   - Call type information in analytics
